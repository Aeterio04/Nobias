# Nobias - AI Trust Engine

Complete bias auditing platform for datasets, ML models, and AI agents.

## Quick Start

1. Install backend: cd backend && pip install -r requirements.txt && pip install unbiased==0.0.0
2. Start backend: cd backend && python start.py
3. Start frontend: npm install && npm run dev

Backend: http://127.0.0.1:8000
Frontend: http://localhost:5173

## Features
- Dataset Auditor: Upload CSV/XLSX, detect bias, get fixes
- Model Auditor: Test ML models for fairness
- Agent Auditor: Test LLM agents for bias

See full documentation in the app.
